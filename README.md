# Docker Ruby Base

Installed necessary package for our ruby application in alpine.

# Usage

```
# Dockerfile
FROM eric4545/docker-ruby-base:2.5.5
```

## Build

- `docker build -t eric4545/docker-ruby-base:latest .`
- `docker build -t eric4545/docker-ruby-base:2.5.5 .`
